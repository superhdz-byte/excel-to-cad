# 節点・要素番号プロット → AutoCAD 自動描画

Excelに記載された有限要素モデル（節点座標・要素接続）を読み込み、AutoCAD 2024 へ COM 経由で自動描画するPythonスクリプトです。

構造物の耐震解析モデル確認図など、FEM前処理の可視化チェックに使用することを想定しています。

---

## 機能

- Excelファイル（`.xlsx`）から節点座標・要素接続情報を読み込み
- AutoCAD 2024 に COM 接続（起動済みの場合はアタッチ、未起動の場合は自動起動）
- 以下を自動描画：
  - **要素線**（I端〜J端の線分）
  - **節点円**（各節点位置に円）
  - **節点番号テキスト**
  - **要素番号テキスト**（要素中点に配置）
- レイヤー自動生成（要素線 / 節点番号 / 要素番号）
- 描画完了後に `.dwg` ファイルとして保存
- エラー・スキップ件数を終了時にレポート

---

## 動作環境

| 項目 | 要件 |
|------|------|
| OS | Windows 10 / 11 |
| Python | 3.8 以上 |
| AutoCAD | 2024（COM対応版） |
| Excel | `.xlsx` 形式 |

> ⚠️ AutoCAD の COM インターフェース（`AutoCAD.Application`）を使用するため、**Windows 環境必須**です。

## Excelファイルの形式

### シート名：`座標`

| 列A（節点番号） | 列B（X座標） | 列C（Y座標） |
|:-:|:-:|:-:|
| 1 | 0.0 | 0.0 |
| 2 | 1000.0 | 0.0 |
| ... | ... | ... |

### シート名：`要素`

| 列A（要素番号） | 列B（I端節点番号） | 列C（J端節点番号） |
|:-:|:-:|:-:|
| 1 | 1 | 2 |
| 2 | 2 | 3 |
| ... | ... | ... |

## 設定

スクリプト冒頭の `# ★ 設定` セクションで以下を変更してください。

```python
EXCEL_PATH = r"F:\path\to\your_model.xlsx"   # 入力Excelのパス
DWG_PATH   = r"F:\path\to\output.dwg"         # 出力DWGのパス

SIZE = 300.0   # 節点円の半径・テキスト高さ（モデル座標系の単位に合わせて調整）
```

### レイヤーカラー（AutoCAD カラーインデックス）

| レイヤー | 変数 | デフォルト |
|--------|------|-----------|
| 要素線 | `COLOR_ELEMENT_LINE` | 3（緑） |
| 節点番号 | `COLOR_NODE_TEXT` | 4（シアン） |
| 要素番号 | `COLOR_ELEM_TEXT` | 1（赤） |

## 使い方

```bash
python autocad_plot_com.py
```

1. Excelファイルから節点・要素データを読み込み
2. AutoCAD に接続（または起動）し、新規図面を作成
3. 要素線 → 節点円・節点番号 → 要素番号 の順で描画
4. `ZoomExtents` で全体表示
5. 指定パスに `.dwg` を保存

---

## 出力例（レイヤー構成）

```
要素線     ← 要素の線分（緑）
節点番号   ← 節点円＋節点番号テキスト（シアン）
要素番号   ← 要素中点テキスト（赤）
```

## 注意事項

- AutoCAD が既に起動している場合、アクティブなセッションに接続します。作業中の図面がある場合は**事前に保存**してください。
- `DWG_PATH` に既存ファイルがある場合は削除してから上書き保存します（AutoCAD で開いている場合は `_new` サフィックスを付けて保存）。
- 大規模モデル（数千要素以上）では描画に時間がかかる場合があります。
